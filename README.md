# ☕ Coffee Shop Sales Dashboard 

## 📊 Project Overview

This project presents an interactive sales dashboard built using Microsoft Excel to analyze transaction data for a coffee and food business. The dashboard provides clear insights into sales performance, revenue trends, and business growth using dynamic visualizations and KPIs.

---

## 🎯 Objectives

* Analyze total transactions, quantity sold, revenue, and profit
* Identify sales distribution across product categories
* Examine weekly transaction trends
* Compare year-over-year revenue and growth
* Evaluate city-wise and category-wise performance
* Visualize monthly sales trends

---

## 🛠️ Tools & Techniques

* Microsoft Excel
* Pivot Tables & Pivot Charts
* Slicers (Year, Month, City)
* Data Aggregation & Calculations
* Growth (%) Analysis
* Data Visualization & Dashboard Design

---

## 📈 Key Features

* KPI cards displaying total revenue, profit, transactions, and quantity sold
* Weekly and monthly sales trend analysis
* Year-over-year revenue comparison with growth calculation
* Revenue and profit comparison across cities and categories
* Interactive filters for dynamic data exploration
* Clean, coffee-themed dashboard design for better readability

## 🔍 Key Insights
